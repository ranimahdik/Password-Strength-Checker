# 🔐 Password Strength Checker

A simple and responsive Password Strength Checker built using HTML, CSS, and JavaScript.  
It evaluates password strength in real-time and provides user-friendly feedback.

---

## 🚀 Features
- Real-time password strength checking
- Visual strength indicator (Weak / Medium / Strong)
- Responsive UI design
- Easy to use interface

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure
password-strength-checker/
│── index.html
│── style.css
│── script.js

---

## 📌 How to Use
1. Enter a password in the input field  
2. Instantly see the strength result  
3. Improve password based on suggestions
